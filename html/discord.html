Obtenir L'application

Vous pouvez télécharger Discord gratuitement sur votre ordinateur ici, ou via le Google Play store ou l'Apple App Store sur les mobiles et les tablettes.

Lucas

Ou si vous ne souhaitez pas télécharger l'application, vous pouvez utiliser notre client Web pour accéder à Discord directement depuis votre navigateur.

Lucas

Créer Votre Compte

Vous pouvez créer votre compte sur la page d'enregistrement de Discord, ou dans l'application.

Lucas
Lucas

Réclamer Votre Compte

Vous pourrez profiter de toutes les fonctions de chat offertes par Discord, mais il est important de réclamer votre compte en vérifiant votre adresse e-mail. Cela vous permettra de conserver votre nom d'utilisateur et votre discriminateur (ces quatre chiffres à côté de votre nom d'utilisateur), et de garder en mémoire tous les serveurs que vous avez rejoints !
Pour réclamer votre compte, vérifiez simplement l'e-mail que vous avez utilisé pour créer votre compte. Si vous ne voyez pas d'e-mail, appuyez simplement sur le bouton Renvoyer dans la bannière orange en haut de l'application :

Lucas

Cliquez sur Vérifier l'e-mail dans l'e-mail reçu, et c'est parti !

Lucas

Vérifier Votre Numéro De Téléphone

En outre, vous pouvez également vérifier votre numéro de téléphone sur votre compte Discord !


Pour commencer le processus de vérification du téléphone, rendez-vous dans l'onglet Mon compte dans vos Paramètres Utilisateur. Puis appuyez sur le bouton Ajouter juste à côté de la section Numéro de téléphone !

Une fois que vous aurez appuyé sur ce bouton, vous pourrez saisir directement votre téléphone, puis appuyer sur "Envoyer" une fois que vous aurez saisi votre numéro ! (Remarque : assurez-vous de sélectionner le bon code pays).


Vous recevrez ensuite un texto sur votre appareil mobile qui contiendra un code à 6 chiffres ! Lorsque vous recevez le code, tapez ces 6 chiffres dans le menu !

Lucas
Lucas
Lucas

Enfin, pour confirmer vos modifications, vous devrez entrer le mot de passe de votre compte, puis appuyer sur le bouton Confirmer.


Votre numéro de téléphone sera alors officiellement vérifié sur votre compte Discord !

Lucas
